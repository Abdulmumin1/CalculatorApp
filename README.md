# compile 
build from source
Clone the repo 
```bash
git clone https://github.com/Abdulmumin1/CalculatorApp.git
cd CalculatorApp

```
Create executable
```bash 

pip install pyinstaller
pyinstaller --windowed --icon CUSTOM/ICON/PATH main.py

```
